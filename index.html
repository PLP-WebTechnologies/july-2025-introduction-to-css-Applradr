WEEK 3: THE BOX MODEL

The Box model gives an html file responsive layout and helps make the page more appealing

Some key elements that make this model effective and are most used are as follows

Margin

This sets the space outside the elements and helps create spacing between cards. you could also use the gap element to add space between cards

Padding

Creates space between elements and around them. helps for clear outline and styling.

Border

Creates a clear line and border that makes the cards and boxes distinct and separated.
